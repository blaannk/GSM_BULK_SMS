# GSM BULK SMS SENDER
Script in python to send mass sms using a gsm modem
## Installation
```
py -m pip install -r requirements.txt
```
## Use
- Connect your GSM modem
- Add your message to message.txt
- Add your numbers to nums.txt (format : E.164 **without the +**)
- ```py main.py```
- Select your port
- Enjoy
## Plateforms
- Windows
- Linux
